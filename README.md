# evmflow-paper-research

Files used in evmflow paper experiments

# 1000 contracts dataset output

Compressed dataset files of the nine evaluated SOTA tool is publicly available at: https://github.com/zerjioang/evmflow-dataset
Decompressing all this files locally will require between 20-30 GB of free space. Dataset includes full JSON trace with:

* input command details
* bytecode
* results
* error details if any
* scan results
* metrics collected during scan
* dot graphs
* svg files from .dot sources
